# Синтаксис

`Print [Текст] - Команда для вывода текста 

все пользовательские данные кроме переменный выделяються {
Пример : Print {Hello world!}

`Chp [название] и End Chp` - Команды которые разделяют главы 
после выполнения всех команд главы начинаеться следущяя глава
или если главы закончлись то программа закрываеться 
чтобы программа не закрывалась при конце не добавляте EndChp
``

`Input $[Переменная] - [Подпись]` - Cтрока ввода , котороя позволяет пользователю задать значение перменной
также можно добавить L[Число] для лимита символов


`$[Переменная] = [Значение] - Устанавливает значение пременной возможны комбинации переменных `$test+$test2` и цифр `1+1`

Интересный факт 1 - в  команте установления значения перменной математика работает по принципу `2+2=22` а в команде Calc работает `2+2=4`

`if [Переменная] = [Значение] - [Действие или команда]` - Команда котороя позволяет производить команды при определенном значении переменной выпоняет команду или действие
одна из самых полезных команд
для нее существует особое действие `Run [Название Главы]`  позволяет переходить из главы в главну и хорошо сочитаеться с Input

`calc $[Переменная]+$[Переменная] - $[Переменная для сохранения результата]`
команда считает математические действия 
поддерживают только цифры

`rpl [Кол-во раз для повторения также можно вставить числовое значение переменной через $] [Команда]` - Команда для повторения команлы

`con ![Команда CleverCMD]` - Команда для выполнения команд CleverOS

`col [Название цвета на англ.]` - Подкоманда Print Которая изменяет цвет текста

`nline` - Подкоманда Print Которая начинет текст идущей послее нее с новой строки

`Wait [Цифровое значение или значение переменной]` - Команда для ожидания перед выполнением действия 1 - 1 секунда

`Exit` - Завершение выполнения

`
`if not found create [Название папки] - Создание папок`

`save [Переменная] in [Here/Другая папке но обязательно в папку проэкта] - [Название файла / можно ставить значение переменных]`

`If pressed [буква или цифра также работают коды по типу Ctrl и Alt возможны комбинации] - [Действие или команда]` - 

`random [Диапазон через запятые ] - [Переменная]` - Рандомно выберает число/слово и тд и записывает значение в переменные

# Инструкции

### Запуск

1. Установите Pyton
2. Установите ZeroShell
3. Напишите код и сохраните его в файл формата TXT
4. Создайте папку проэкта в `Scripts` и переместите TXT туда
5. Запустите Shell и напишите !Run [Название файла с указанием разширения пример: Calc.txt

### Разработка

для разработки приложений лучше всего использовать блокнот Windows 

# Примеры приложений

### Калькулятор

chp Main
print {добро пожаловать в калькулятор" Nline "Введите первое и второе число для сложения чисел }

Input $1num - {1 число}
Input $2num - {2 число}
Calc $1num + $2num - $result

print {Результат -} $result

End chp
chp pvt

Input $Yn - {Начать заново?(y/n) }

if $Yn = {Y} - Run chp main
if $Yn = {N} - Exit

### Блокнот

Chp main

Input $text - {Пишите свой текст здесь}
If pressed {ctrl+s} - Input $namefile - {Введите имя и расширение файла}
if not found create {Files}
save $text in {files} - $namefile
print{файл сохранен}

end chp
chp Yn

Input $Yn - {Начать заново?(y/n) }

if $Yn = {Y} - Run chp main
if $Yn = {N} - Exit

end chp
